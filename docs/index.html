Getting Started

1

Download

Grab the latest StopwatchOverlay-vX.X.X.zip from the Releases page.

2

Extract

Unzip to a folder of your choice, for example C:\Tools\StopwatchOverlay.

3

Run

Double-click StopwatchOverlay.exe to launch the controller. No installation needed.

4

.NET Runtime

Windows 11 (24H2+) includes the required runtime. On older versions, install the .NET 8.0 Desktop Runtime.
